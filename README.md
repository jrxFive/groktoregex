# GrokToRegex - Convert logstash patterns to regex
GrokToRegex is a simple command line utility that converts known grok alias to its corresponding regex value.

## Installation
To install GrokToRegex, use go get
```
go get github.com/jrxfive/groktoregex
```

## Usage
Once built groktoregex only needs two flags, '--grok' and '--source'.
Source is a folder that contains logstash grok names and their values. The format used is:
```
"<GROK_NAME>" = "<VALUE>"
```

Add custom grok values to the provided file or create a new file. GrokToRegex will use a glob
and pick up any files within the given folder.

## Example
```
./groktoregex --pattern "%{HOSTPORT}" --lookup patterns/

# %{HOSTPORT} is actually "%{IPORHOST}:%{POSINT}"
```
